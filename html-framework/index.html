Framework

A reusable 466 x 466 round-screen starter for StopWatch apps. It gives new prototypes the same shell, A/B button model, pointer coordinate mapping, feedback helpers, and embed mode used by the hardware previews.

Interaction model

The framework starts with a simple mode-based app loop that can be replaced by a real prototype while keeping the same hardware-oriented input surfaces.

  • Press A to enter or reset the running state.
  • Press B to cycle modes while idle.
  • Pointer input is converted into 466 x 466 device coordinates.

Hardware parity

This template is intentionally close to the app pages: the shell, controls, feedback gates, audio tone helpers, vibration calls, and embedded preview contract match the StopWatch workflow.

  • Use `?embed=1` for a clean 466 x 466 preview surface.
  • Keep confirmed browser constants ready for firmware migration.
  • Replace the canvas drawing logic without changing the page contract.
